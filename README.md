# 🪔 Diwali Sales Analysis

# 📊 Overview :

This project analyzes retail sales data during the Diwali festival to understand customer purchasing behavior and generate business insights. The focus is on identifying trends, top-performing segments, and opportunities for improving sales strategies.

# 📁 Dataset :
The dataset contains customer transaction data with features such as:
 - Customer demographics (age, gender, state)
 - Product details
 - Purchase amount and sales information
   
# 🛠️ Tools & Technologies :
 - Python (Pandas, NumPy)
 - Matplotlib / Seaborn
   
# ⚙️ Project Steps :
 - Data Loading – Imported dataset using Pandas
 - Exploratory Data Analysis (EDA) – Analyzed trends and distributions
 - Data Cleaning – Handled missing values, removed irrelevant columns, and corrected data types
 - Data Visualization – Created charts to understand patterns and relationships
   
# 📈 Results :
 - Identified key customer segments contributing to sales
 - Found high-performing products and regions
 - Observed purchasing trends based on age and gender
   
# ✅ Conclusion :

The project demonstrates how data analysis can help businesses understand customer behavior and make informed decisions to improve sales and marketing strategies.
